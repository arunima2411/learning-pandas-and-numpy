# Learning Pandas and NumPy

## Project Overview

This repository contains hands-on exercises and mini-projects completed while learning Pandas and NumPy for data analysis in Python.

The notebooks demonstrate data extraction, data manipulation, and feature extraction techniques using real datasets.

## Files

### Data Files
- `Countries.csv` - Dataset containing country-related information used for data analysis and extraction tasks.
- `anime.csv` - Dataset used for practicing data cleaning, exploration, and feature extraction.

### Notebooks
- `Data_Extraction.ipynb` - Examples of extracting and filtering data using Pandas.
- `Pandas_Feature_Extraction.ipynb` - Feature extraction and data transformation techniques using Pandas.

## Topics Covered

- Reading CSV files
- Exploring datasets
- Selecting rows and columns
- Filtering data
- Data extraction techniques
- Feature extraction
- Working with DataFrames
- Basic NumPy operations
- Data manipulation using Pandas

## How to Run

1. Clone the repository

```bash
git clone https://github.com/arunima2411/learning-pandas-and-numpy.git
```

2. Install required libraries

```bash
pip install pandas numpy
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run the notebooks:
   - Data_Extraction.ipynb
   - Pandas_Feature_Extraction.ipynb
